JS Dragon Slayer

Overview

I really wanted to create a fun project that would push me to use javascript in a way that I usually don't. There were lots of things that I was thinking about doing but eventually settled on a game. Game logic was interesting to take on and I encountered problems I had no idea I would encounter. Like how to tilt the probability of a user winning while still making it possible for the other side to win. Not only was the logic a nice little challenge to tackle but so was the design. I had to ask myself tons of question about how everything would interact on the screen. Making the characters and creating a miniature world of my own was a bunch of fun and am happy with the result.

Bitcoin Price Check

Overview

With this project, I wanted to tackle react js and as an added challenge, I also decided to work in an API. The React JS part is fairly simple, it just renders everything that happens in the API call function. I decided to use a Bitcoin price checker because the API was simple enough that I wouldn't run into too many problems with React, only calling the price (depending on currency) and the currency symbol.

Event Landing Page

Overview

This was a landing page for a client's event. The design was provided by a designer which left the implementation up to me. HTML and CSS (and a little bit of JS on occasion) are used to make these types of landing pages for clients. Often clients use software that requires up to code up landing pages with all the code being in one document. Mobile and desktop views are a must for all web dev projects and this is no exception. These landing pages are a fun way to start something from scratch and keep up with the fundamentals of clean DRY code.

Sketch Mockup

Overview

For this project I wanted to use Sketch to mock up an app. I wanted to get familiar with how Sketch works and understand the principles and best practices. The versatility and effeciency of Sketch allowed me to quickly mock up designs before finally settling on the final design. I wanted to focus on creating a clean and simple design with not to many bells and whistles since I would be building my knowledge of Sketch. Overall I loved the experience and Sketch is going to be my go to mockup software in the future.
