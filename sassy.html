Sass-y Site

Overview

This project was built as a testing ground for Sass. I've used it in the past but really wanted to take a crack at it again. I made sure to take a mobile-first approach to this project so I can start with even less code and then build on it. Then jquery was used for the navigation to create a smooth show and a functional auto scroll when a link is clicked.
